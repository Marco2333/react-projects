## 个人博客管理端

### 技术栈

```
react + antd + react-router + ES6/7 + webpack + express
```

## Usage

### 安装
```
git clone https://github.com/Marco2333/react-projects.git

cd react-projects/admin

npm install
```

#### 开发模式：
```
npm run dev
```

#### 产品模式：
```
npm run build
```
